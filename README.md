# Tulsi College Website

A complete, production-ready, fully responsive website for Tulsi College (India).

## Features

- ✅ Fully Responsive Design (Mobile, Tablet, Desktop)
- ✅ Modern UI/UX with clean design
- ✅ Dark Mode Support
- ✅ Smooth Animations
- ✅ SEO Optimized
- ✅ Form Validation
- ✅ Image Gallery with Lightbox
- ✅ Course Search & Filter
- ✅ Faculty Filter by Department
- ✅ Events & Notices Board
- ✅ Google Maps Integration

## Pages

1. **Home** - Hero section, mission/vision, courses overview, achievements, quick links
2. **About Us** - History, principal's message, management, infrastructure
3. **Courses** - All courses with search and filter functionality
4. **Admissions** - Eligibility, documents, fees structure, online application form
5. **Facilities** - Library, labs, sports, hostel, transportation
6. **Faculty** - List of all faculty members with department filter
7. **Events & Notices** - Upcoming events and downloadable notices
8. **Gallery** - Photo gallery with lightbox
9. **Contact** - Contact form, address, phone, email, Google Maps

## How to Use

1. Open `index.html` in a web browser
2. All pages are linked and functional
3. Forms include validation
4. Dark mode can be toggled from the navigation bar
5. Mobile menu is available on smaller screens
